# 📱 핸드폰에서 LiveJourney 사용하기

## 🌐 방법 1: 모바일 브라우저에서 바로 사용 (가장 쉬움!)

### 📲 사용 방법

**1. URL 공유받기**
```
https://aesthetic-liger-71c7e1.netlify.app
```

**2. 모바일 브라우저에서 접속**
- Chrome, Safari, Samsung Internet 등 어떤 브라우저든 가능
- URL을 입력하거나 링크를 클릭

**3. 바로 사용!**
- 앱처럼 작동합니다
- 설치 필요 없음
- 언제든지 접속 가능

---

## 📲 방법 2: 홈 화면에 추가 (앱처럼 사용하기)

### Android (Chrome)

**1. 사이트 접속**
```
https://aesthetic-liger-71c7e1.netlify.app
```

**2. 메뉴 열기**
- 우측 상단 점 3개(⋮) 클릭

**3. "홈 화면에 추가" 선택**
- 또는 "앱 설치" 선택

**4. 이름 확인 후 추가**
- 기본 이름: "LiveJourney"
- 추가 클릭

**5. 홈 화면에서 실행!**
- 아이콘이 생성됨
- 일반 앱처럼 사용 가능
- 전체 화면으로 실행됨 (주소창 없음)

---

### iPhone (Safari)

**1. 사이트 접속**
```
https://aesthetic-liger-71c7e1.netlify.app
```

**2. 공유 버튼 클릭**
- 하단 중앙의 공유 아이콘 (□↑) 클릭

**3. "홈 화면에 추가" 선택**
- 아래로 스크롤하여 찾기
- "Home Screen에 추가" 클릭

**4. 이름 확인 후 추가**
- 기본 이름: "LiveJourney"
- 우측 상단 "추가" 클릭

**5. 홈 화면에서 실행!**
- 아이콘이 생성됨
- Safari 없이 독립 실행
- 전체 화면으로 표시

## 🔧 방법 3: PWA 기능 강화 (선택사항)

현재 상태에서도 작동하지만, PWA 기능을 추가하면 더 앱처럼 사용 가능합니다.

### 추가 가능한 기능:
1. **오프라인 지원** - 인터넷 없이도 일부 기능 사용
2. **푸시 알림** - 새 글 알림 등
3. **백그라운드 동기화** - 오프라인에서 업로드 → 온라인 시 자동 업로드
4. **설치 프롬프트** - "앱 설치하시겠습니까?" 팝업

현재는 기본 PWA로 작동 중이므로 **홈 화면 추가만으로도 충분**합니다!
