# ⚡ 빠른 시작 가이드

## 🚀 가장 쉬운 실행 방법

### 방법 1: 웹 브라우저에서 확인 (가장 빠름! ⚡)

**`웹실행.bat`** 파일을 더블클릭하세요!

→ 브라우저에서 바로 확인 가능 (네이티브 기능 제외)

또는:
```bash
cd web
npm run dev
```

### 방법 2: Android Studio에서 실행 (네이티브 기능 포함)

**`START_HERE.bat`** 파일을 더블클릭하세요!

→ 자동으로 빌드 → 동기화 → Android Studio 열기

또는:
```bash
cd web
npm run quick:run
```

## 📱 Android Studio에서 실행

1. **Gradle 동기화 대기** (최초 1회만, 2-5분)
2. **Run 버튼 클릭** (또는 `Shift + F10`)
3. **에뮬레이터 또는 실제 디바이스 선택**
4. **앱 실행!**

## ⚠️ 에뮬레이터가 실행되지 않는 경우

### 🎯 가장 빠른 해결: 실제 안드로이드 폰 사용 (추천!)

**에뮬레이터보다 실제 폰이 훨씬 빠르고 간단합니다!**

1. **안드로이드 폰 설정:**
   - 설정 → 휴대전화 정보 → 빌드 번호 **7번 연속 탭**
   - 설정 → 개발자 옵션 → **USB 디버깅** 켜기

2. **USB로 PC에 연결**

3. **Android Studio에서 Run → 실제 폰 선택**

→ 자세한 가이드: `빠른해결_실제디바이스.md` 참고

### 🔧 에뮬레이터 설정이 필요한 경우

1. **Android Studio → Tools → Device Manager**
2. **Create Device** 클릭
3. **Pixel 5** 또는 **Pixel 6** 선택
4. **시스템 이미지 다운로드** (API 33 또는 34)
5. **Finish**

→ 자세한 가이드: `에뮬레이터_설정가이드.md` 참고
