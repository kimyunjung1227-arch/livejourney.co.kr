# 🌐 Capacitor 앱을 웹 브라우저에서 확인하기

## ⚡ 가장 빠른 방법

Capacitor 앱은 기본적으로 **웹 앱**이므로, 브라우저에서 바로 확인할 수 있습니다!

### 실행 방법

```bash
cd web
npm run dev
```

또는

```bash
cd web
npm start
```

→ 브라우저가 자동으로 열립니다 (`http://localhost:3000`)

## ✅ 장점

- ⚡ **가장 빠른 확인 방법**
- 🔧 **개발 중 실시간 수정 확인**
- 📱 **반응형 디자인 확인 가능**
- 🎨 **모든 UI/UX 확인 가능**

## ⚠️ 제한사항

다음 기능들은 웹 브라우저에서 작동하지 않습니다:
- 📸 카메라 (파일 선택은 가능)
- 📍 GPS 위치 (브라우저 위치 권한은 가능)
- 📱 네이티브 알림
- 💾 파일 시스템 접근

## 🔄 네이티브 기능이 필요한 경우

네이티브 기능(카메라, GPS 등)을 테스트하려면:

```bash
cd web
npm run quick:run
```

→ Android Studio에서 실제 디바이스나 에뮬레이터로 실행

## 💡 개발 워크플로우 추천

1. **일반 개발/디자인 확인:** 웹 브라우저 (`npm run dev`)
2. **네이티브 기능 테스트:** Android Studio (`npm run quick:run`)

---

**결론:** Capacitor 앱은 웹 브라우저에서 바로 확인할 수 있습니다! 🎉
