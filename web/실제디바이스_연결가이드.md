# 📱 실제 안드로이드 디바이스 연결 가이드

## ⚡ 가장 빠른 방법 (에뮬레이터 대신)

### 1단계: 안드로이드 폰 설정

1. **개발자 옵션 활성화:**
   - 설정 → 휴대전화 정보 → 빌드 번호 **7번 연속 탭**
   - "개발자가 되었습니다!" 메시지 확인

2. **USB 디버깅 활성화:**
   - 설정 → 개발자 옵션 → **USB 디버깅** 켜기

3. **PC에 연결:**
   - USB 케이블로 안드로이드 폰을 PC에 연결
   - 폰에서 "USB 디버깅 허용" 팝업 → **허용** 체크

### 2단계: Android Studio에서 실행

1. Android Studio 실행
2. **Run** 버튼 클릭 (또는 `Shift + F10`)
3. **연결된 디바이스 목록**에서 **실제 폰** 선택
4. 앱이 폰에 설치되고 실행됩니다!

## ✅ 장점

- ⚡ **에뮬레이터보다 훨씬 빠름**
- 🎯 **실제 기기에서 테스트 가능**
- 💾 **RAM 부족 문제 없음**
- 🔋 **설정이 간단함**

## 🔍 디바이스가 보이지 않는 경우

### Windows 드라이버 설치

1. **Google USB Driver** 다운로드:
   - Android Studio → Tools → SDK Manager
   - SDK Tools 탭
   - **Google USB Driver** 체크 → Apply

2. **디바이스 관리자 확인:**
   - Windows 검색 → "디바이스 관리자"
   - "기타 디바이스" 또는 "ADB Interface" 확인
   - 드라이버 업데이트 필요 시 Google USB Driver 사용

### ADB 명령어로 확인

```bash
cd web/android
adb devices
```

연결된 디바이스 목록이 보이면 성공!

## 💡 팁

- **무선 디버깅** (Android 11+):
  - 개발자 옵션 → 무선 디버깅 활성화
  - IP 주소로 연결 가능
