## 📐 페이지 구조

### 1. Navigation (고정 헤더)
- 투명한 배경 + 블러 효과
- 로고 + 메뉴 + CTA 버튼
- 스크롤 시 항상 표시

### 2. Hero (첫 인상)
**목표:** 3초 안에 가치 전달
- **헤드라인:** "당신의 소중한 여정이 헛되지 않도록"
- **서브헤드:** 명확한 가치 제안
- **CTA:** 2개 버튼 (주요 + 보조)
- **Social Proof:** 사용자 수 + 평점
- **Visual:** 앱 목업 (실시간 화면)

### 3. Problem (공감대)
**목표:** "나도 그래!" 공감 유발
- 3가지 실제 문제 사례
- 이모지로 감정 표현
- 짧고 명확한 스토리

### 4. Solution (해결책)
**목표:** "이게 답이구나!" 확신
- 실시간 LIVE 강조
- 3가지 핵심 가치
- 비주얼 + 텍스트 조합

### 5. How It Works (작동 원리)
**목표:** "이렇게 쉽구나!" 이해
- 3단계 프로세스
- 화살표로 흐름 표시
- 각 단계별 아이콘

### 6. Features (기능)
**목표:** "이것도 되는구나!" 발견
- 6가지 핵심 기능
- 그리드 레이아웃
- 아이콘 + 제목 + 설명

### 7. Testimonials (후기)
**목표:** "믿을 만하구나!" 신뢰
- 3개의 실제 후기
- 별점 + 텍스트 + 프로필
- 구체적인 경험담

### 8. Final CTA (최종 전환)
**목표:** "지금 시작해야지!" 행동
- 강력한 헤드라인
- 이메일 입력 폼
- 신뢰 배지 (보안, 스팸 없음)

### 9. Footer (정보)
- 로고 + 링크들
- 법적 고지
- Copyright

## 💡 전문적인 요소들

### 1. 타이포그래피
```
Heading: 800 weight (Extra Bold)
Subheading: 600-700 weight
Body: 400-500 weight
Size: 명확한 계층 (h1: 3.5rem, h2: 2.5rem)
Line Height: 1.75 (편안한 가독성)
```

### 2. 간격 (Spacing)
```
Sections: 5rem (80px)
Cards: 2rem padding
Grid Gap: 1.5-2rem
일관된 리듬감
```

### 3. 그림자 (Shadows)
```
Subtle: 0 1px 2px
Medium: 0 4px 6px
Large: 0 10px 15px
XLarge: 0 20px 25px
```

### 4. 둥근 모서리 (Border Radius)
```
Small: 0.375rem (6px)
Medium: 0.5rem (8px)
Large: 0.75rem (12px)
XLarge: 1rem (16px)
2XLarge: 1.5rem (24px)
Full: 9999px (pill shape)
```

### 5. 호버 효과
```
Transform: translateY(-2px/-4px)
Shadow: 강화
Transition: 0.3s ease
Border Color: 변경
```

## 🎯 UX 최적화

### Visual Hierarchy (시각 계층)
1. **헤드라인** (가장 큼, 볼드)
2. **서브헤드** (중간, 세미볼드)
3. **본문** (작음, 레귤러)
4. **CTA** (색상으로 강조)

### F-Pattern Reading (F자 읽기)
- 왼쪽 정렬 유지
- 중요 정보는 왼쪽 상단
- CTA는 자연스러운 시선 흐름

### White Space (여백)
- 충분한 패딩 (2rem)
- 섹션 간 간격 (5rem)
- 텍스트 블록 간 여유
- "숨쉬는" 디자인

### Color Psychology (색상 심리)
- **파란색**: 신뢰, 안정, 전문성
- **오렌지**: 활력, 즉시성, 행동
- **회색**: 중립, 정보, 배경
- **흰색**: 깔끔함, 명료함
