## 🎨 업로드 버튼 기술

### HTML
```html
<div class="nav-item upload">
  <div class="nav-icon-large"></div>
</div>
```

### CSS
```css
/* 원 배경 */
.nav-icon-large {
  width: 56px;
  height: 56px;
  background: linear-gradient(135deg, #00BCD4, #0097A7);
  border-radius: 50%;
  position: relative;
}

/* + 기호 */
.nav-icon-large::before {
  content: '+';
  position: absolute;
  top: 50%;              /* 상단에서 50% */
  left: 50%;             /* 왼쪽에서 50% */
  transform: translate(-50%, -50%);  /* 정확히 중앙으로 */
  font-size: 32px;
  font-weight: 300;
  line-height: 1;
}
```

### 원리
1. **position: relative** - 부모 요소 기준점
2. **::before** - 가상 요소로 + 생성
3. **absolute** - 절대 위치
4. **transform: translate(-50%, -50%)** - 정확한 중앙
