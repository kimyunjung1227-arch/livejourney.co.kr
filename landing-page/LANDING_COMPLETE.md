## 🚀 실행 방법

### 방법 1: Python 서버 (현재 실행 중)
```bash
cd landing-page
python -m http.server 8000
```

**브라우저에서 열기:**
```
http://localhost:8000
```

### 방법 2: 배치 파일 실행
```bash
cd landing-page
./START_LANDING.bat
```

### 방법 3: 파일 직접 열기
```
landing-page/index.html
```
파일을 더블클릭하면 기본 브라우저에서 열립니다.

## 📂 파일 구조

```
landing-page/
├── index.html              # 메인 HTML (실제 앱 mockup 포함)
├── styles.css              # 스타일링 (핸드폰 UI)

├── images/                 # 실제 여행 사진
│   ├── sample1.jpg        # 제주 성산일출봉

│   └── sample3.jpg        # 추가 이미지
```

## 🔧 추가 커스터마이징

### 이미지 변경하기
1. 원하는 이미지를 `landing-page/images/` 폴더에 복사
2. `index.html`에서 이미지 경로 수정:
```html
<img src="images/your-image.jpg" alt="설명" />
```

### 텍스트 변경하기
1. `index.html` 열기
2. 사용자 이름, 위치, 게시물 내용 수정
3. 저장 후 새로고침

### 색상 변경하기
1. `styles.css` 열기
2. `:root` 섹션에서 CSS 변수 수정:
```css
:root {
  --primary: #2563eb;      /* 메인 색상 */
  --accent: #f59e0b;       /* 강조 색상 */
}
```
