### 2. ✅ 플러스 버튼 원 정중앙 정렬
**완벽한 중앙 정렬**

```css
.nav-icon-large {
  font-size: 28px;           /* 아이콘 크기 조정 */
  font-weight: 400;          /* 두께 */
  width: 56px;               /* 원 크기 */
  height: 56px;
  line-height: 0;            /* 수직 중앙 ⭐ */
  padding: 0;                /* 여백 제거 ⭐ */
  margin: 0;                 /* 마진 제거 ⭐ */
  display: flex;
  align-items: center;       /* 세로 중앙 */
  justify-content: center;   /* 가로 중앙 */
}
```

**핵심 변경:**
- `line-height: 0` - 텍스트 라인 높이 제거
- `padding: 0` - 내부 여백 완전 제거
- `margin: 0` - 외부 여백 완전 제거

### 3. ✅ 헤더 단순화
**컬러 박스 제거, 깔끔한 디자인**

```css
Before:
background: linear-gradient(135deg, #00BCD4, #0097A7);
color: white;

After:
background: white;
color: var(--gray-900);
border-bottom: 1px solid var(--gray-200);
```

**변경 사항:**
- 파란색 그라데이션 배경 → 흰색 배경
- 흰색 텍스트 → 검은색 텍스트
- 하단 테두리 추가 (구분선)

## 🔧 플러스 버튼 수정 디테일

### 문제점
```
원의 크기: 56px × 56px
아이콘: + (add_circle)
문제: 아이콘이 위쪽으로 치우침
```

### 해결책
```css
line-height: 0;    /* 텍스트 기준선 제거 */
padding: 0;        /* 모든 내부 여백 제거 */
margin: 0;         /* 모든 외부 여백 제거 */
```

### 결과
```
✅ + 아이콘이 원의 정중앙에 위치
✅ 수직/수평 완벽 정렬
✅ 시각적으로 균형 잡힌 디자인
```
